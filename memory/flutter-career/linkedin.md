🚀 How I built an EKYC package used by 2M+ users across Bangladesh's biggest fintech platforms.

When Polygon Technology asked me to build a reusable EKYC module that could work across bKash, City Bank, and Prime Bank — I had 3 constraints:

1. Must work across 500+ Android device types
2. Must handle spotty 2G/3G network conditions
3. Must be integration-ready in under 2 weeks per client

→ Modular BLoC state machine (each step: NID scan, liveness check, data parse = independent state)

→ Skeleton loading UI (perceived speed wins over raw speed)

Result: <1% failure rate. 60% faster client integration. 3 banks, one package.

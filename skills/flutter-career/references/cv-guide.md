# CV Templates & Guides

## ATS-Friendly CV Structure

```
[Name]
[Title] | [City, Country] | [Phone] | [Email] | [LinkedIn] | [GitHub]
---
SUMMARY
One-paragraph value proposition: X years Flutter/Dart experience, 
specialize in [domains], delivered [metric] impact.

---
EXPERIENCE

**Company Name** | Job Title
Month/Year – Present (or Month/Year – Month/Year if past)

• [Verb] + [What] + [Result/Metric]
• [Verb] + [What] + [Result/Metric]
• [Verb] + [What] + [Result/Metric]

**Company Name** | Job Title
Month/Year – Month/Year

• [Verb] + [What] + [Result/Metric]
• [Verb] + [What] + [Result/Metric]

---
SKILLS

Languages: Dart, Kotlin, Swift, Python, JavaScript
Frameworks: Flutter, React Native, Jetpack Compose
Tools: Firebase, GitHub Actions, Codemagic, Riverpod, BLoC, GetX
CI/CD: GitHub Actions, Fastlane, Codemagic

---
EDUCATION

Degree | University | Year

---
PROJECTS (optional)

[Project Name] | [Link] | [Tech Stack]
[One-line description + impact]
```

## Impact Verb Bank

| Category | Verbs |
|----------|-------|
| Built/Created | Designed, Developed, Architected, Built, Created, Established, Launched |
| Improved | Optimized, Enhanced, Improved, Increased, Reduced, Streamlined |
| Led/Managed | Led, Coordinated, Managed, Directed, Supervised, Mentored |
| Delivered | Delivered, Released, Shipped, Deployed, Launched, Implemented |
| Collaborated | Collaborated, Partnered, Worked with, Aligned with |

## Common Mistakes to Fix

❌ "Responsible for building features"
✅ "Built and shipped 12 customer-facing features, reducing load time by 40%"

❌ "Worked on EKYC module"
✅ "Developed EKYC verification module handling 50K+ daily KYC submissions with 99.9% uptime"

❌ "Used Flutter and Firebase"
✅ "Architected Flutter + Firebase real-time chat system supporting 10K concurrent users"

## LinkedIn CV vs Regular CV

- LinkedIn: First person optional, more narrative
- CV: Third person preferred, bullet-heavy, metrics-focused
- Both: Same skills, same experience, different framing

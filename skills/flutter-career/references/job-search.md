## In-Demand Skills (2024-2025)

### Must-Have
- Flutter + Dart
- State Management (Riverpod, Bloc, GetX)
- REST APIs / GraphQL
- Firebase / Supabase
- Git + GitHub

### Nice-to-Have
- Kotlin / Swift (native bridging)
- CI/CD (GitHub Actions, Codemagic)
- BLoC pattern
- Clean Architecture
- Widget testing

### Emerging
- Fuchsia OS awareness
- Embedded Flutter
- AR/VR integration
- AI/ML with Flutter

# ai_assistant_config

[Boltu ⚡](https://github.com/SHAJED99) - Personal AI Assistant Workspace for Shajedur Rahman Panna (Panna)

## Purpose

This repository stores my complete workspace memory, configuration, and operational data. Anytime a new session starts, I load from here to instantly know:

- Who Panna is (profile, preferences, skills)
- Who I am (Boltu, identity, capabilities)
- Current projects and tasks
- Technical environment
- Recent decisions and context

## Repository Structure

```
ai_assistant_config/
├── config/           # Identity, preferences, environment
├── workspace/        # User profile, assistant state, projects, tasks
├── memory/          # Summary, patterns, reusable context
├── prompts/         # Startup and specialized prompts
├── changelog/       # Version history
├── backups/         # Snapshots
├── restore/         # Restore guide
└── docs/            # Commands documentation
```

## Quick Commands

- **Push data** → Save and push workspace
- **Sync workspace** → Update context
- **Backup now** → Create snapshot
- **Restore from repo** → Load latest state
- **Show workspace** → Summarize current state

## For Reconfiguration

If Boltu needs to be reconfigured from scratch:
1. Clone this repo
2. Load config files
3. Authenticate GitHub CLI
4. Continue from saved state
